# מערכת ניהול נתונים - דירוג ציונות (v2)

מערכת ניהול נתונים מתקדמת לניהול רשומות URL במערכת דירוג ציונות.

## מבנה הפרויקט

```
ranking-v2/
├── public/                          # קבצים ציבוריים (נגישים מהדפדפן)
│   ├── index.html                  # עמוד ראשי
│   ├── assets/
│   │   ├── css/
│   │   │   ├── main.css           # עיצוב ראשי
│   │   │   └── components.css     # עיצוב רכיבים
│   │   └── js/
│   │       ├── app.js             # אפליקציה ראשית
│   │       ├── api.js             # תקשורת עם API
│   │       ├── table.js           # ניהול טבלה
│   │       ├── inline-edit.js     # עריכה ישירה (מודול נפרד)
│   │       └── modals.js           # ניהול מודלים
│   └── api/                        # API endpoints
│       ├── records.php            # CRUD operations
│       ├── summaries.php          # ניהול סיכומים
│       └── ai/                    # AI operations
│           └── relevance.php
├── src/                            # קוד מקור (לא נגיש ישירות)
│   ├── config/
│   │   ├── database.php
│   │   └── api_key.php
│   ├── lib/
│   │   └── url_fetcher.php
│   └── utils/
│       └── helpers.php
├── database/
│   ├── schema.sql
│   └── migrations/
└── README.md
```

## תכונות

- תצוגת רשומות בטבלה עם עיצוב מודרני
- חיפוש וסינון מתקדם
- עריכה ישירה של קטגוריה וסוג ארגון (תוקן!)
- עריכת רשומות
- יצירת סיכומים אוטומטיים
- דירוג רלוונטיות AI
- ייצוא נתונים ל-JSON

## התקנה

1. ודא שהקבצים נמצאים בתיקיית הפרויקט שלך
2. ודא שהגדרות מסד הנתונים ב-`src/config/database.php` נכונות
3. ודא שטבלת `ranking_urls` קיימת במסד הנתונים (הרץ את `database/schema.sql` אם צריך)

### הפעלה עם PHP Built-in Server:

```bash
cd ranking-v2/public
php -S localhost:8000
```

ואז פתח בדפדפן: `http://localhost:8000`

## שינויים מהגרסה הקודמת

1. מבנה מסודר יותר עם הפרדה בין קבצים ציבוריים וקוד מקור
2. JavaScript מפוצל למודולים נפרדים
3. תיקון בעיית הלחיצה על תאי עריכה ישירה
4. API מאורגן יותר
