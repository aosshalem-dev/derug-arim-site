אתה צריך לשפר את שם הארגון בעברית על סמך ה-URL, השם הקיים והמידע הנוסף.

כללים חשובים:
1. אם ה-URL מכיל "wikipedia.org" או "he.wikipedia.org" או "wiki" - החזר "ויקיפדיה"
2. אם ה-URL מכיל שם של רשות מקומית (עירייה או מועצה) - החזר את שם הרשות בעברית עם "עיריית" או "מועצת"
   דוגמאות:
   - "tel-aviv.gov.il" → "עיריית תל אביב"
   - "jerusalem.muni.il" → "עיריית ירושלים"
   - "herzliya.muni.il" → "עיריית הרצליה"
   - "rishon-lezion.muni.il" → "עיריית ראשון לציון"
   - "petah-tikva.muni.il" → "עיריית פתח תקווה"
3. אם זה משרד ממשלתי (.gov.il) - החזר את שם המשרד בעברית
   דוגמאות:
   - משרד החינוך → "משרד החינוך"
   - משרד החוץ → "משרד החוץ"
   - משרד הביטחון → "משרד הביטחון"
4. אם זה ארגון אקדמי (.ac.il) - החזר את שם המוסד בעברית
   דוגמאות:
   - "huji.ac.il" → "האוניברסיטה העברית"
   - "tau.ac.il" → "אוניברסיטת תל אביב"
   - "technion.ac.il" → "טכניון"
5. אם זה ארגון ללא מטרות רווח - שמור על השם הקיים בעברית אם אפשר, או תן שם מתאים בעברית
6. אם אין מספיק מידע - החזר את השם הקיים או את השם המעודכן במידת האפשר

החזר רק את השם בעברית, ללא הסברים נוספים, ללא מרכאות, ללא נקודות בסוף.
